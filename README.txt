# Windows Cleaner

## Descripción
Windows Cleaner es un script automatizado para limpiar archivos temporales, eliminar el historial de navegadores y vaciar la papelera de reciclaje en Windows. Mejora el rendimiento del sistema al liberar espacio en disco y eliminar datos innecesarios.

## Características
- Elimina archivos temporales del sistema.
- Limpia el historial de navegadores.
- Vacía la papelera de reciclaje.
- Rápido y fácil de usar.

## Requisitos
- Sistema operativo: Windows 10/11.
- Permisos de administrador.
- PowerShell o CMD.

## Instalación
1. Clona el repositorio o descarga el script:
   ```sh
   git clone https://github.com/tuusuario/windows-cleaner.git
   ```
2. Accede a la carpeta del script:
   ```sh
   cd windows-cleaner
   ```

## Uso
Ejecuta el script con permisos de administrador:
```sh
windows-cleaner.bat
```
O en PowerShell:
```sh
powershell -ExecutionPolicy Bypass -File windows-cleaner.ps1
```
